## 测量建议

- **胸围**：沿腋下水平测量胸部最丰满处一周
- **腰围**：沿腰部最细处水平测量一周
- **臀围**：沿臀部最丰满处水平测量一周

> 提示：不同品牌、不同版型的服装尺码可能存在差异，建议参考商品详情页的具体尺码表。如有疑问可咨询在线客服。
