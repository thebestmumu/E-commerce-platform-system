# 材质与洗涤说明

## 常见面料保养

### 纯棉 (Cotton)
- **特点**：透气吸汗、柔软舒适
- **洗涤**：不超过 40°C 温水洗涤，使用中性洗涤剂
- **禁忌**：不可漂白、不可高温烘干
- **晾晒**：阴凉处自然晾干，避免暴晒导致褪色

### 羊毛 (Wool)
- **特点**：保暖性好、柔软弹性
- **洗涤**：建议干洗或手洗，使用羊毛专用洗涤剂
- **禁忌**：不可机洗、不可拧绞、不可暴晒
- **晾晒**：平铺阴干，不可悬挂（易变形）

### 丝绸 (Silk)
- **特点**：光滑柔软、透气性好
- **洗涤**：冷水手洗，使用丝绸专用洗涤剂
- **禁忌**：不可碱性洗涤剂、不可机洗、不可暴晒
- **晾晒**：阴凉处阴干，低温熨烫

### 涤纶 (Polyester)
- **特点**：抗皱耐磨、易干
- **洗涤**：可机洗，水温不超过 40°C
- **禁忌**：不可高温熨烫
- **晾晒**：可正常晾晒

### 牛仔 (Denim)
- **特点**：耐磨耐穿、越穿越有型
- **洗涤**：首次冷水浸泡固色，反面洗涤
- **禁忌**：不宜频繁洗涤，不要用热水
- **晾晒**：反面晾晒，避免褪色

## 通用保养建议

- 深色衣物首次洗涤前用盐水浸泡 30 分钟可固色
- 内衣、泳装建议手洗
- 阅读衣物内侧的洗标标识，按标识说明洗涤
- 不同颜色的衣物分开洗涤，避免染色
