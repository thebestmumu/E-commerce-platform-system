## 测量建议

- **脚长测量**：赤脚站立在纸上，标记脚后跟和最长的脚趾，测量两点距离
- **脚宽测量**：测量脚掌最宽处的距离
- 建议在下午测量脚长，此时脚部略微肿胀，尺码更准确
- 袜子厚度也会影响穿着效果，建议穿平时常穿的厚度的袜子测量

> 提示：不同品牌的鞋型略有差异，运动鞋、皮鞋、靴子的尺码选择可能不同，建议参考具体商品的页面说明。
