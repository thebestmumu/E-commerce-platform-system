# 标准配送时效

## 配送时效标准

根据收货地址的不同，标准配送时效如下：

- **同城配送**：1-2 天送达
- **省内城市**：2-3 天送达
- **跨省城市**：3-5 天送达
- **偏远地区**（新疆、西藏、青海、甘肃）：5-10 天送达

## 特殊说明

1. 新疆、西藏等偏远地区因地理原因，配送时间可能延长至 10-15 天，请耐心等待
2. 如遇法定节假日或电商大促期间（618、双11等），配送时效可能顺延 1-2 天
3. 下单后一般 48 小时内发货，预售商品以商品页标注时间为准
4. 发货后可在「我的订单」中查看实时物流信息

## 配送范围

全国范围（港澳台地区除外）均可配送，部分特殊区域（如海岛、军事管理区等）可能需要额外时间。
