# 七天无理由退货

## 适用条件

自签收之日起 **7 天内**，商品符合以下条件即可申请无理由退货：

1. 商品保持完好，未经使用、未经洗涤
2. 不影响二次销售
3. 原包装、配件、赠品齐全
4. 防伪标识、标签完好未撕毁

## 不适用商品

以下商品不支持七天无理由退货：

- **定制商品**（如定制服装、个性化印制品）
- **贴身内衣**（内衣裤、泳装等）
- **虚拟商品**（充值卡、会员服务、电子券）
- **生鲜食品**（水果、生鲜肉类、熟食等）
- **个人卫生用品**（牙刷、剃须刀等）
- **已拆封的音像制品、软件类商品**

## 申请方式

1. 登录账号，进入「我的订单」
2. 找到对应订单，点击「申请售后」
3. 选择「退货」，填写退货原因
4. 上传商品照片（无需拆封的各个角度）
5. 提交申请，等待客服审核

> 无理由退货的运费由买家承担。建议购买运费险，退货时可获得运费赔付。
