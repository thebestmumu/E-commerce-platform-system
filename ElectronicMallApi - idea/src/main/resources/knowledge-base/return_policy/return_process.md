# 退货流程

## 完整退货步骤

### 第一步：提交申请
在「我的订单」详情页点击「申请售后」，选择「退货」及退货原因

### 第二步：上传凭证
- 质量问题：上传商品照片/视频（含瑕疵部位）
- 错发/漏发：上传收到的商品照片和快递单
- 无理由退货：上传商品整体照片

### 第三步：等待审核
客服将在 **1-2 小时**内完成审核，审核结果将通过站内信和短信通知

### 第四步：寄回商品
- **审核通过**后按系统提供的退货地址寄回商品
- 可在售后页面选择「上门取件」（部分城市支持）
- 请使用快递寄回，建议保留快递单号

### 第五步：商家签收验收
商家收到商品后进行验收，确认完好后进入退款流程

### 第六步：完成退款
验收通过后 **48 小时内**完成退款

## 运费承担

| 退货原因 | 运费承担方 | 说明 |
|---------|-----------|------|
| 质量问题 | 商家承担 | 系统自动生成上门取件 |
| 错发/漏发 | 商家承担 | 联系客服处理 |
| 七天无理由 | 买家承担 | 如有运费险可赔付 |
| 商品与描述不符 | 商家承担 | 需提供截图/证据 |
