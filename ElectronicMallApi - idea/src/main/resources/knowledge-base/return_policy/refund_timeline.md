# 退款到账时间

## 各支付方式到账时间

| 支付方式 | 到账时间 |
|---------|---------|
| 支付宝余额 | 即时到账 |
| 支付宝余额宝 | 即时到账 |
| 支付宝绑定的银行卡 | 1-3 个工作日 |
| 微信零钱 | 即时到账 |
| 微信绑定的银行卡 | 1-3 个工作日 |
| 银联卡 | 3-7 个工作日 |
| 花呗 | 额度实时恢复，资金 1-3 个工作日 |

## 退款进度查询

1. 在「我的订单」中找到退款中的订单
2. 点击「查看退款详情」查看当前进度
3. 系统会通过短信和站内信通知退款每个阶段的完成情况

## 异常情况处理

- **超过预期时间未到账**：请联系在线客服查询
- **退款金额有误**：联系客服核实，提供订单号和商品信息
- **退款失败**：检查绑定的银行卡状态是否正常，或联系客服更换退款方式

> 提示：如商家在验收退货后 48 小时内未操作退款，系统将自动触发退款流程，保障您的权益。
