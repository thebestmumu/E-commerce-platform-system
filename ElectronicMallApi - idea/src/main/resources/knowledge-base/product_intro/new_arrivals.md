# 新品上市

## 新品更新频率

- 平台每周一、周四更新新品
- 每月进行一次新品主题大上新

## 新品标识

在商品列表中，新品商品会带有「新品」角标标识，方便快速识别。

## 新品专区

在首页导航栏中有「新品上市」专区入口，汇集了近期上新的所有商品，可按分类筛选浏览。

## 新品权益

- 新品上市前 3 天享受 **9 折尝鲜价**
- 购买新品可获 **双倍积分**
- 新品可叠加使用 **新人优惠券**

> 提示：新品库存有限，热门新品可能很快售罄。关注商品页面可设置「到货通知」。
