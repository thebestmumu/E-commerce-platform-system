# 账户安全

## 安全防护建议

1. **密码安全**
   - 使用复杂密码，避免使用生日、手机号等易猜信息
   - 不在其他平台使用相同的密码
   - 定期更换密码，建议每 3 个月更新一次

2. **登录环境**
   - 不在公共电脑上自动保存密码
   - 不连接不安全的公共 Wi-Fi 进行支付
   - 退出登录时请点击「安全退出」

3. **账号绑定**
   - 建议同时绑定手机号和邮箱
   - 开启二次验证（如支持）
   - 定期查看登录设备记录

4. **防诈骗提醒**
   - 平台客服不会以任何理由索要您的密码或验证码
   - 不要点击来历不明的链接
   - 任何要求转账到个人账户的行为都是诈骗

## 账号异常处理

如发现账号存在以下异常：
- 异常登录（非本人操作的登录记录）
- 订单异常（出现非本人操作的订单）
- 信息被修改（密码、手机号、邮箱被篡改）

请立即：
1. 尝试通过「忘记密码」重置密码
2. 联系在线客服冻结账号
3. 检查绑定的手机号和邮箱

## 设备管理

在「个人中心-账号安全」中可查看所有登录过的设备，可手动移除不认识的设备。如遇设备丢失，建议立即移除该设备的登录权限并修改密码。
