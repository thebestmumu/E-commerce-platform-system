# 密码重置

## 找回密码步骤

### 手机号找回
1. 在登录页面点击「忘记密码」
2. 输入注册手机号
3. 获取短信验证码
4. 输入验证码验证身份
5. 设置新密码（6-20 位字母+数字组合）
6. 确认新密码
7. 重置成功，返回登录页面

### 邮箱找回
1. 在登录页面点击「忘记密码」
2. 选择「邮箱找回」
3. 输入注册邮箱
4. 登录邮箱查收验证邮件
5. 点击邮件中的重置链接
6. 设置新密码
7. 重置成功

## 密码安全要求

- 密码长度：6-20 位
- 建议包含：大写字母、小写字母、数字、特殊符号中至少三种
- 不能与用户名或手机号相同
- 不能是连续或重复字符（如 123456、aaaaaa）

## 账号被锁定

如连续 **5 次**输入错误密码，账号将被临时锁定 **30 分钟**。锁定期间无法登录，请等待解锁后再试，或通过「忘记密码」立即重置密码。

> 提示：如遇到手机号不再使用或收不到验证码的情况，请联系在线客服进行人工身份验证后重置密码。
