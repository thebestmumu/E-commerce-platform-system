# 修改订单

## 可修改的内容

订单提交后，在不同状态可修改的内容不同：

### 待支付状态可修改
- 收货地址（仅限修改同城范围内的地址）
- 收货人联系方式
- 发票信息
- 备注/留言

### 待发货状态可修改
- 收货人电话
- 备注信息

### 已发货状态
- 不可修改
- 如需改地址，请联系客服尝试联系快递公司转寄

## 修改操作步骤

1. 进入「我的订单」
2. 找到需要修改的订单
3. 点击「修改订单」
4. 修改对应信息
5. 保存确认

## 无法修改的情况

- 商品颜色、尺码、数量等无法直接修改
- 如需变更商品信息，建议取消订单后重新下单
- 已发货订单的收货地址无法直接修改

> 提示：修改收货地址仅在未发货前有效，请在下单时仔细核对地址信息。
