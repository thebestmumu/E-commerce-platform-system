# 订单状态说明

## 订单状态详解

一个订单从创建到完成，会经历以下状态：

### 待付款
- 订单已提交成功，等待买家付款
- 超时 **30 分钟**未支付，订单自动取消
- 可在「我的订单」中点击「继续支付」

### 已支付/待发货
- 买家已完成付款
- 商家正在备货
- 承诺 **48 小时内**发货
- 此阶段可以申请取消订单（部分商品除外）

### 已发货
- 商品已发出，物流单号已生成
- 可在订单详情页查看实时物流信息
- 发货后不可自主取消订单

### 已签收
- 买家已确认签收
- 从签收日起开始计算 **7 天无理由退货**时效
- 可选择「确认收货」或等待系统自动确认

### 已完成
- 订单交易成功
- 可对商品进行评价和晒图
- 评价后可获得积分奖励

### 已取消
- 订单已被取消
- 已支付的金额会按原路退回

## 系统自动确认收货

如买家未主动确认收货，系统会在商家发货后 **15 天**（普通商品）或 **30 天**（海外购/定制商品）后自动确认收货。

> 提示：收到商品后请及时检查，确认无误后点击「确认收货」。如有问题请在 7 天内申请售后。
