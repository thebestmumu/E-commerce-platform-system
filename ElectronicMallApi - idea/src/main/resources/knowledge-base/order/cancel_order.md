# 取消订单

## 取消规则

根据订单状态不同，取消规则如下：

| 订单状态 | 能否取消 | 退款方式 |
|---------|---------|---------|
| 待付款 | 可取消 | 无支付行为无需退款 |
| 已支付/待发货 | 可取消 | 全额原路退回 |
| 已发货 | 需拒收后取消 | 退款需扣除运费 |
| 已签收 | 不可直接取消 | 需申请售后退货 |

## 取消操作步骤

### 待付款订单
1. 进入「我的订单」
2. 找到待付款订单
3. 点击「取消订单」
4. 选择取消原因并确认

### 已支付/待发货订单
1. 进入「我的订单」
2. 找到对应订单
3. 点击「申请取消」
4. 选择退款原因
5. 审核通过后自动退款

### 已发货订单
已发货订单如需取消：

1. 在物流派送时明确告知快递员拒收
2. 同时在「我的订单」中申请取消
3. 商品退回商家验货后办理退款

## 退款到账时间

退款金额将按原支付方式退回，具体到账时间请参考「退款到账时间」文档。

> 提示：部分特殊商品（如定制商品、生鲜）在下单后不支持取消订单，详情请查看商品页面的说明。
