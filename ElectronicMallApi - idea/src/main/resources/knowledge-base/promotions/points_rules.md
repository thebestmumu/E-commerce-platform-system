# 积分规则

## 积分获取

| 获取方式 | 积分数量 | 说明 |
|---------|:-------:|------|
| 购物消费 | 消费金额 × 1 | 每消费 1 元获得 1 积分 |
| 会员倍率加成 | 额外积分 | 银卡 1.2 倍、金卡 1.5 倍、钻石 2 倍 |
| 商品评价 | 10-50 积分 | 晒图评价额外赠送 |
| 每日签到 | 随机 1-5 积分 | 连续签到 7 天额外奖励 20 积分 |
| 参与活动 | 视活动而定 | 大促活动、问卷调查等 |

## 积分使用

| 用途 | 兑换标准 | 说明 |
|-----|:-------:|------|
| 兑换优惠券 | 100 积分 = 5 元券 | 需在积分商城兑换 |
| 兑换商品 | 视商品而定 | 积分+现金兑换 |
| 抵扣现金 | 100 积分 = 1 元 | 结算时选择积分抵扣 |
| 兑换抽奖 | 50 积分/次 | 可抽取优惠券或实物奖品 |

## 积分有效期

- 积分有效期为 **12 个月**
- 积分按获取时间先后到期（先进先出原则）
- 即将到期的积分会在「个人中心-我的积分」中显示

## 积分查询

- 在「个人中心-我的积分」中查看积分明细
- 每笔积分变动都有详细记录（来源、时间、数量）
- 积分商城可查看可兑换的优惠券和商品

> 提示：积分不可转让，不可兑换现金。退货时，该笔订单获得的积分将被扣除。
