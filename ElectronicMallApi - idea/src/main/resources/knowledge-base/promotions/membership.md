# 会员权益

## 会员等级体系

平台会员共分为四个等级，根据年度累计消费金额自动升降级：

| 等级 | 年消费门槛 | 升级奖励 |
|:---:|:---------:|:--------:|
| 普通会员 | 无门槛（默认） | - |
| 银卡会员 | 累计满 1,000 元 | 50 积分 + 满 100 减 10 券 |
| 金卡会员 | 累计满 5,000 元 | 200 积分 + 满 200 减 30 券 |
| 钻石会员 | 累计满 20,000 元 | 500 积分 + 满 500 减 100 券 |

## 各等级权益对比

| 权益 | 普通 | 银卡 | 金卡 | 钻石 |
|:---:|:---:|:---:|:---:|:---:|
| 购物积分倍率 | 1 倍 | 1.2 倍 | 1.5 倍 | 2 倍 |
| 生日礼包 | - | 满 100 减 20 | 满 200 减 50 | 满 500 减 150 |
| 免费包邮 | 满 99 | 满 69 | 满 39 | 不限金额 |
| 专属客服 | - | - | 优先接入 | 专属 VIP 通道 |
| 退货运费险 | - | 赠送 | 赠送 | 赠送 |
| 新品预约 | - | - | 优先 | 优先 |

## 等级有效期

- 会员等级有效期为 **1 年**（从升级之日起计算）
- 到期后按上一年度的消费金额重新评定等级
- 如连续两年达到该等级门槛，等级可延续

> 提示：在「个人中心-会员中心」中可查看当前等级、成长进度和即将到期的权益。
